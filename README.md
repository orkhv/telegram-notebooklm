## Telegram → NotebookLM (локально)

Простой одностраничный инструмент для локальной конвертации экспортированных чатов Telegram (`result.json`) в один или несколько удобных Markdown‑файлов, оптимизированных под загрузку в NotebookLM / LLM.

Данные никуда не отправляются — вся обработка происходит в браузере на вашей машине.

---

### Возможности

- **Единый поток сообщений**: все выбранные чаты собираются в одну непрерывную ленту.
- **Удобный формат**: каждая строка имеет вид  
  `[YYYY-MM-DD HH:MM] Автор: текст`.
- **Работа с большими архивами**: при большом объёме текст разбивается на части примерно по 300 000 слов (`*_part_N.md`).
- **Полностью локально**: используется только статический `index.html`, без серверной части.

---

### Как получить `result.json` в Telegram Desktop

1. Откройте **Telegram Desktop**.
2. Зайдите в **Настройки → Дополнительно → Экспорт данных из Telegram**.
3. Выберите нужные чаты (или все чаты) и формат **JSON**.
4. Запустите экспорт и дождитесь окончания.
5. В папке экспорта возьмите файл `result.json`.

---

### Как пользоваться этой страницей

**Вариант 1: GitHub Pages (готовый хостинг)**

- Откройте страницу проекта:  
  `https://orkhv.github.io/telegram-notebooklm/`
- Перетащите файл `result.json` в зону загрузки или выберите его вручную.
- Выберите нужные чаты (если нужно).
- Нажмите кнопку **«Скачать единый файл»**.
- Полученный файл(ы) `*_part_N.md` загрузите в NotebookLM / другую LLM.

---

### Конфиденциальность

- Файл `result.json` **не отправляется** ни на какие серверы.
- Вся логика обработки и конвертации выполняется в браузере в виде обычного клиента.
